= Ardoonga Engine

Ardoonga pomoże Ci stworzyć internetową tekstową grę RPG. 
Nie musisz konfigurować żadnych serwerów, ani za nic płacić. 
Wszystko zrobi się automatycznie.

Zazwyczaj zaczynamy od wybrania rasy i klasy swojej postaci. Dodajmy kilka z nich.

NOTE: Ardoonga nie odróżnia ras i klas. 
Z punktu widzenia silnika klasy posiadają wiedzę, której nie można zdobyć w inny sposób. Sprawdź dokument <<wiedza>>, jeśli chcesz wiedzieć więcej.

Ale najpierw zacznijmy od najważniejszego. Gdzie wrzucać swoje pliki i jak je napisać ?
